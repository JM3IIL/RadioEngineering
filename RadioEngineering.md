## 等方性アンテナ
利得の基準

放射電力P[W]のとき、d[m]離れた点の電界強度
```math
E = \frac{\sqrt{30P}}{d} [V/m]
```

## 微小ダイポールアンテナ

放射電力P[W]のとき、d[m]離れた点の電界強度
```math
E = \frac{\sqrt{45P}}{d} [V/m]
```

## 半波長ダイポールアンテナ
放射電力P[W]のとき、d[m]離れた点の電界強度
```math
E = \frac{\sqrt{49.2P}}{d} ≒ \frac{7\sqrt{P}}{d} [V/m]
```
