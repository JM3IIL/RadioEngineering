覚える
- 1/4波長垂直接地ANTの放射抵抗 : 36Ω
- 半波長ダイポールANTの放射抵抗 : 73Ω

- 放射電力 : $P [W]$
- アンテナ入力電流 : $I [A]$
- 放射抵抗 : $R_r [Ω]$
- 実効抵抗 : $R_E [Ω]$ (実効抵抗=放射抵抗+導体抵抗+接地抵抗)
- 接地抵抗 : $R_g [Ω]$
- 放射効率 : $η$ (イータ)

 
$P = I^2 R_r$
$η = \frac{R_r}{R_E} ≒ \frac{R_r}{R_r + R_g}$
