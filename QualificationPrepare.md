R06_01_A01 アンテナ 利得 指向性 受信電力
- 可逆定理 フリスの伝達公式 配列指向係数 バビネの原理
- 相対利得 = 絶対利得-2.15 [dB]
- 半波長ダイポール:利得=等方性アンテナ1.64倍 絶対利得=2.15 [dB]
- 微小ダイポール:利得=等方性アンテナ1.5倍 絶対利得=約1.76 [dB]

- 不整合の時の反射損 $M$ (真数)
- アンテナの動作利得 $G/M$
- 反射係数 $\varGamma$
- $M = 1/(1-|\varGamma|^2)$

R06_01_A03 微小ダイポール アンテナ利得 実効面積

R06_01_A05 電波の偏波
- 直線偏波 : 電波の進行方向に垂直な面上で、互いに直交する方向の電界成分の位相差が0 [rad]又はπ [rad]
- 円偏波 : 電波の進行方向に垂直な面上で、互いに直交する方向の電界成分の位相差がπ/2 [rad]で振幅が等しい
- 楕円偏波 : 電波の進行方向に垂直な面上で、互いに直交する方向の電界成分の位相差がπ/2 [rad]で振幅が異なる
- 軸比 $E_1 / E_2$ (真数) : 楕円偏波の長軸方向の電界強度 $E_1$ と短軸方向の電界強度 $E_2$ との比
 - 軸比の大きさが1に近い 円偏波に近い
 - 軸比の大きさが∞に近い 直線偏波に近い

R05_07_2_A08 方形導波管
- 方形導波管の長辺寸法aが波長の半分以下では電波は進行しない
- 遮断波長 $λ_c = \frac{1}{\sqrt{(\frac{m}{2a})^2 + (\frac{n}{2b})^2}} [m]$
- X方向の波の数 m
- Y方向の波の数 n
- 一般に $TE_{mn}$ の形式で表す
- 導波管内を進行可能な最も周波数の低い伝送モード　m = 1、n = 0 : $TE_{10}$ モード
- 管内波長 $λ_g = \frac{λ}{\sqrt{1 − (\frac{λ}{2a})^2}} [m]$
- $TE_{10}$モードの位相速度 $v_p = \frac{c}{\sqrt{1 − (\frac{λ}{2a})^2}} [m/s]$
- $TE_{10}$モードの群速度 $v_g = c \sqrt{1 − (\frac{λ}{2a})^2} [m/s]$
